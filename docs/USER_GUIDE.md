## 🚀 Premiers Pas

### Inscription

1. **Visiter** https://getyourshare.com
2. **Cliquer** sur "S'inscrire"
3. **Choisir** votre rôle :
   - 🏪 **Marchand** : Vous vendez des produits/services
   - 👤 **Influenceur** : Vous promouvez des produits
   - 💼 **Commercial** : Vous gérez des comptes
4. **Remplir** le formulaire :
   - Email professionnel
   - Mot de passe sécurisé (min. 8 caractères)
   - Nom de l'entreprise (pour marchands)
5. **Vérifier** votre email
6. **Compléter** votre profil

### Connexion

```
URL : https://getyourshare.com/login
Email : votre-email@example.com
Mot de passe : ••••••••
```

**Mot de passe oublié ?**
1. Cliquer sur "Mot de passe oublié"
2. Entrer votre email
3. Vérifier votre boîte mail
4. Suivre le lien de réinitialisation

### Choisir un Plan d'Abonnement

| Plan | Prix/mois | Idéal pour |
|------|-----------|-----------|
| **Free** | 0€ | Découverte, petits projets |

## 🏪 Guide Marchand

### Tableau de Bord

**Accès :** Dashboard → Vue d'ensemble

Visualisez vos indicateurs clés :
- 💰 **Revenu total** du mois
- 📊 **Ventes** (nombre + montant)
- 👥 **Influenceurs actifs**
- 📈 **Taux de conversion**

### Gestion des Produits

#### Ajouter un Produit

1. **Aller** à `Produits` → `Ajouter un produit`
2. **Remplir** les informations :
   - Nom du produit
   - Description (min. 50 caractères)
   - Prix (€)
   - Catégorie
   - Stock disponible
   - Images (max 5)
3. **Définir** le taux de commission (ex: 10%)
4. **Cliquer** sur "Publier"

**Conseils :**
- ✅ Utilisez des images de haute qualité (1200x1200px)
- ✅ Descriptions détaillées augmentent les conversions
- ✅ Taux de commission compétitif attire plus d'influenceurs

#### Modifier un Produit

1. **Aller** à `Produits` → Cliquer sur le produit
2. **Modifier** les champs
3. **Sauvegarder** les changements

#### Désactiver un Produit

Produits → ⋮ Menu → "Désactiver"

### Créer une Campagne

**Campagne = Ensemble de produits promus sur une période**

1. **Aller** à `Campagnes` → `Nouvelle campagne`
2. **Nommer** la campagne (ex: "Black Friday 2024")
3. **Choisir** :
   - Date de début
   - Date de fin
   - Budget (optionnel)
   - Taux de commission spécial
4. **Sélectionner** les produits inclus
5. **Lancer** la campagne

**Exemple de campagne réussie :**
```
Nom : Soldes d'Été 2024
Période : 1-31 juillet
Budget : 5000€
Commission : 15% (au lieu de 10%)
Produits : 20 articles mode
Résultat : 15K€ de ventes, 2250€ de commissions versées
```

### Gérer les Influenceurs

#### Voir les Influenceurs Actifs

`Influenceurs` → Liste de tous les influenceurs qui promeuvent vos produits

Informations visibles :
- Nom / Pseudo
- Nombre d'abonnés
- Taux de conversion
- Ventes générées
- Commission due

#### Approuver un Influenceur

Certains marchands choisissent de valider manuellement les influenceurs.

1. `Influenceurs` → `Demandes en attente`
2. Consulter le profil
3. Accepter ou Refuser

### Analytics & Rapports

#### Vue d'ensemble

`Analytics` → Dashboard complet :
- 📊 Graphique des ventes (30 derniers jours)
- 🎯 Top 10 produits
- 👥 Top 10 influenceurs
- 🌍 Ventes par pays
- 📱 Ventes par appareil (mobile/desktop)

#### Rapports Avancés

`Rapports` → Générer des rapports personnalisés :

**Types de rapports :**
1. **Ventes** : Détail des transactions
2. **Commissions** : Commissions versées par influenceur
3. **Clics** : Analyse des clics par lien
4. **Produits** : Performance par produit

**Export :**
- 📄 CSV
- 📊 Excel
- 📰 PDF

**Planification :**
- Recevoir des rapports automatiquement par email
- Fréquence : quotidien, hebdomadaire, mensuel

### Paiements

#### Configurer les Paiements

`Paramètres` → `Paiements` :
- Stripe (carte bancaire)
- PayPal
- Virement bancaire (SEPA)
- CMI (Maroc)

#### Voir les Transactions

`Transactions` → Historique complet :
- Date
- Influenceur
- Produit
- Montant
- Commission
- Statut (en attente, payé, échoué)

#### Payer les Commissions

`Paiements` → `Commissions à payer`

Options :
- Paiement manuel (virement)
- Paiement automatique (configurer dans Paramètres)

## 👤 Guide Influenceur

### Tableau de Bord

**Accès :** Dashboard → Statistiques influenceur

Vos KPIs :
- 💰 **Commission gagnée** ce mois
- 🔗 **Clics** sur vos liens
- 🛒 **Conversions** (achats)
- 📈 **Taux de conversion**

### Marketplace

#### Explorer les Produits

`Marketplace` → Parcourir tous les produits disponibles

**Filtres :**
- Catégorie (mode, tech, beauté, etc.)
- Prix
- Taux de commission (min/max)
- Marchand

**Recherche :**
Utilisez la barre de recherche pour trouver des produits spécifiques.

#### Ajouter au Panier

1. **Cliquer** sur un produit
2. **Consulter** les détails
3. **Cliquer** sur "Promouvoir"
4. Le produit est ajouté à votre sélection

### Créer un Lien d'Affiliation

#### Méthode 1 : Depuis le Marketplace

1. **Sélectionner** un produit
2. **Cliquer** sur "Créer un lien"
3. **Copier** le lien généré
4. **Partager** sur vos réseaux sociaux

#### Méthode 2 : Depuis Mes Liens

1. `Mes Liens` → `Nouveau lien`
2. **Choisir** une campagne ou un produit
3. **Personnaliser** (optionnel) :
   - Paramètres UTM
   - Nom du lien (pour suivi interne)
4. **Générer** le lien

**Exemple de lien :**
```
https://getyourshare.com/r/abc123xyz
```

### Partager vos Liens

**Bonnes pratiques :**

✅ **Instagram Story** :
- Swipe up avec votre lien
- Mention du produit
- Code promo exclusif

✅ **YouTube** :
- Lien dans la description
- Mention verbale dans la vidéo
- Pin commentaire

✅ **TikTok** :
- Lien en bio
- Mention dans la vidéo
- #ad #sponsored

✅ **Blog** :
- Article de revue
- Comparatif produits
- Tutoriel d'utilisation

⚠️ **Conformité** : Toujours mentionner qu'il s'agit d'un lien affilié.

### Suivre vos Performances

#### Statistiques par Lien

`Mes Liens` → Cliquer sur un lien :
- 📊 Clics quotidiens
- 🛒 Conversions
- 💰 Commission gagnée
- 🌍 Origine du trafic (Instagram, YouTube, etc.)

#### Top Produits

Identifiez vos produits les plus performants pour optimiser vos promotions.

### Demander un Paiement

`Paiements` → `Demander un retrait`

**Conditions :**
- Minimum 50€ de commission
- Vérification d'identité complétée

**Méthodes de paiement :**
- Virement bancaire (SEPA)
- PayPal
- Wise (international)

**Délais :**
- Traitement : 3-5 jours ouvrés
- Virement : 1-2 jours

## 💼 Guide Commercial

### Rôle du Commercial

Gérer les relations clients :
- Prospects
- Leads
- Comptes existants

### Gestion des Leads

#### Ajouter un Lead

1. `Leads` → `Nouveau lead`
2. **Remplir** :
   - Nom du contact
   - Entreprise
   - Email
   - Téléphone
   - Source (LinkedIn, événement, etc.)
   - Statut (nouveau, contacté, qualifié, converti)
3. **Enregistrer**

#### Pipeline de Vente

`Pipeline` → Visualisation Kanban :
- **Nouveau** : Lead entrant
- **Contacté** : Premier contact établi
- **Qualifié** : Lead intéressé
- **Proposition** : Devis envoyé
- **Négociation** : Discussion en cours
- **Gagné** : Client converti ✅
- **Perdu** : Opportunité perdue ❌

**Drag & Drop** pour déplacer les leads entre les étapes.

### Suivi des Opportunités

#### Créer une Opportunité

1. `Opportunités` → `Nouvelle`
2. **Lier** au lead ou compte
3. **Définir** :
   - Montant estimé
   - Date de clôture prévue
   - Probabilité (%)
4. **Suivre** l'évolution

#### Activités

Enregistrer toutes vos interactions :
- 📞 Appels téléphoniques
- 📧 Emails
- 🤝 Réunions
- 📝 Notes

### Rapports Commerciaux

`Rapports` → `Commercial` :
- Pipeline value
- Taux de conversion
- Durée moyenne du cycle de vente
- Performance par commercial

## ❓ FAQ

### Questions Générales

**Q : GetYourShare fonctionne-t-il en mode SaaS ?**  
R : Oui, c'est une solution 100% cloud. Aucune installation requise.

**Q : Puis-je essayer gratuitement ?**  
R : Oui, le plan Free est disponible sans carte bancaire.

**Q : Les données sont-elles sécurisées ?**  
R : Oui, chiffrement SSL, conformité RGPD, sauvegardes quotidiennes.

### Questions Marchands

**Q : Quel est le taux de commission recommandé ?**  
R : Entre 5% et 20% selon votre marge. 10-15% est la moyenne.

**Q : Puis-je vérifier les influenceurs avant qu'ils promeuvent ?**  
R : Oui, activez l'option "Approbation manuelle" dans les paramètres.

**Q : Comment éviter la fraude aux clics ?**  
R : Notre système détecte automatiquement les clics suspects.

### Questions Influenceurs

**Q : Combien de temps avant de recevoir mes commissions ?**  
R : Généralement 3-5 jours après la demande de retrait.

**Q : Y a-t-il un minimum de followers requis ?**  
R : Non, tous les créateurs de contenu sont bienvenus.

**Q : Puis-je promouvoir plusieurs marchands ?**  
R : Oui, il n'y a pas de limite.

### Questions Techniques

**Q : L'application fonctionne-t-elle hors ligne ?**  
R : Oui (PWA), certaines fonctionnalités sont disponibles hors ligne.

**Q : Existe-t-il une application mobile ?**  
R : L'interface est responsive et installable comme PWA sur mobile.

**Q : Y a-t-il une API ?**  
R : Oui, disponible à partir du plan Pro.
