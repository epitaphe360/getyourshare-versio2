## 🔍 ÉTAPE 4 : TRACKING AUTOMATIQUE DES VENTES

### Comment le système sait que la vente vient du live ?

**3 méthodes d'attribution (par ordre de priorité) :**

#### Méthode 1 : Cookie de tracking (Plus fiable)

```
Quand le client clique sur getyourshare.ma/r/PARFUM001:

1. Système crée un cookie:
   {
     "tracking_code": "PARFUM001",
     "influencer_id": "uuid_influenceur",
     "live_session_id": "uuid_session_live",
     "expires": "30 jours"
   }

2. Client achète (maintenant ou dans 30 jours)

3. Au moment de la vente, système vérifie:
   - Cookie existe? ✅
   - Live encore actif? ✅ (ou max 2h après)
   - Attribution à l'influenceur ✅
   - Bonus +5% commission appliqué ✅
```

#### Méthode 2 : Code promo

```
Client entre code: LIVE20

Système check:
1. Code existe? ✅
2. Code lié à quelle session live?
   └─ Session ID: uuid_session_live
3. Session live de quel influenceur?
   └─ Influenceur ID: uuid_influenceur
4. Attribution ✅
```

#### Méthode 3 : Paramètres UTM (backup)

```
URL partagée: getyourshare.ma/r/PARFUM001
Contient automatiquement:
?utm_source=PARFUM001
&utm_medium=live_shopping
&utm_campaign=tiktok_live_301125

Système track et attribue ✅
```
