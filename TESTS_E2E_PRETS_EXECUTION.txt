❓ TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

Problème: "Cypress could not verify that this server is running"
Solution: 
  1. Assurez-vous que npm start est lancé dans /frontend
  2. Attendez le message "webpack compiled successfully"
  3. Puis lancez npm run cy:run dans un autre terminal

Problème: "Tests not found"
Solution:
  1. Vérifier que cypress/ecommerce-dashboard-phases-2-3-4.cy.js existe
  2. Vérifier que cypress.config.js est correct
  3. Relancer npm run cy:run

Problème: "cy:run command not found"
Solution:
  1. Vérifier package.json contient les scripts
  2. Relancer npm install
  3. Vérifier que Cypress est installé: npm list cypress

Problème: Tests échouent
Solution:
  1. Vérifier que le frontend démarre sans erreurs
  2. Vérifier que tous les composants sont importés
  3. Consulter les logs Cypress pour détails
